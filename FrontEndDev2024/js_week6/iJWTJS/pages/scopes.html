Variable Scopes in JS

There are scopes in Javascript. Where a variable is only accessible within the same block scope.

Globally declared variable 
 const x = 0;
console.log(x);

(() => {
  console.log(x);
})();
Output 
 0
0
Locally declared variable 
 (() => {
  const y = 0;
})();

console.log(y);
Output 
 scopes_example.js:12 Uncaught ReferenceError: y is not defined
    at scopes_example.js:12:13
Storing Variables in a reference object

In JS there is pass-by-value and pass-by-reference. Most things are pass-by-value but objects are not. We can therefore store a value in the global window object provided by the browser. 

 (() => {
  window.y = 1;
})();
console.log(window.y);

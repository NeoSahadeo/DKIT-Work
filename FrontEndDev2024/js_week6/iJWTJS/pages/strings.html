Strings in JS

Strings can be manipulated in Javascript. Strings act as an array-list of characters. (Array-lists are a muttable data structure).

Strings extend the base `string` class.

String.prototype

Type in `String.prototype` in the console to list out the base string class attributes. 

 String.prototype
Concatenating strings using the '+' operator 
 const x = "Hello";
const y = "World";

console.log(x + y);

// Output: HelloWorld
Slicing a string using the slice method. 
 const x = "Hello";
const y = "World";

console.log(x.slice(0, 4));

// Output: Hell
